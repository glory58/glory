# Évaluation Git & GitHub de Glory

Projet réalisé dans le cadre du TP Git.

## Qu’est-ce que Git ?
Git est un outil de gestion de versions qui permet de suivre les modifications d’un projet, de revenir en arrière et de travailler de manière organisée.

## Qu’est-ce que GitHub ?
GitHub est une plateforme en ligne qui permet d’héberger des projets Git et de les partager ou de collaborer avec d’autres personnes.
